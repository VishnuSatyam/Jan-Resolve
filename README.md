# 🏛️ Jan Resolve

**Jan Resolve** is a modern, web-based citizen grievance platform that empowers users to register, track, and manage public complaints efficiently. Built with a clean and interactive UI, it bridges the gap between citizens and government services through transparency and responsiveness.

---

## 📌 About the Project

Jan Resolve aims to simplify the process of lodging civic complaints and tracking their resolution. With AI-inspired complaint categorization and real-time status updates, it ensures citizens stay informed while helping authorities manage grievances systematically.

> _Empowering citizens. Improving governance. One complaint at a time._

---

## ✨ Features

- 🔐 **User Authentication** — Secure login and registration for citizens
- 📝 **Complaint Registration** — Easy-to-use forms for filing public grievances
- 🤖 **AI-Inspired Categorization** — Smart classification of complaints by type/department
- 📊 **Real-Time Status Tracking** — Live updates on complaint progress and resolution
- 🖥️ **Interactive UI** — Clean, modern interface for seamless user experience
- 🔍 **Complaint Management** — View, filter, and manage submitted complaints

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (comes with Node.js)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VishnuSatyam/Jan-Resolve.git
   cd Jan-Resolve
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```

The production-ready files will be output to the `dist/` folder.
